Game Rules

Each falling block will rest when it hits a surface. If the blocks form a horizontal line along one row, that row will disappear and you will be rewarded with 100 points. Each time you earn points, the speed at which the blocks fall will increase, gradually making the game more challenging. If the blocks touch the top of the screen, then it is game over.

Tips

Use the rotate button to align the blocks to better suit the gap you're trying to fill.

Use the preview window to see which block is coming next and plan how you should react.

Try to start moving the block immediately as it spawns to safely navigate the screen

Use the best block to destroy lines to make the next line easier.
